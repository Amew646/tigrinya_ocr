Intro::
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install cmake vim build-essential git

Connect to WiFi::
sudo vi /etc/wpa_supplicant.conf
```
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
country=US
update_config=1

network={
    ssid="Home Wifi"
    psk="mypassword"
}
```
